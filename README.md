Program Pengolahan Nilai Mahasiswa.
Input Data: Mahasiswa memasukkan jumlah data (maksimal 30) sesuai poin A.
Struktur Data: NIM disimpan dalam array 1D nim, sedangkan nilai Tugas, UTS, dan UAS disimpan dalam matriks 2D nilai.
Menu Operasi: Tersedia fitur Tampilkan, Tambah, Ubah, Hapus (dengan sistem geser array), dan Hitung Nilai.
Pengolahan: Menghitung rata-rata per mahasiswa dan per kategori nilai, serta mencari nilai tertinggi/terendah.
Fitur Rekursif: Implementasi fungsi faktorial dan fibonacci berdasarkan jumlah mahasiswa saat ini.
